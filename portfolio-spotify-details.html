Spotify Web Player Home Clone

A clone of the Spotify Web Player built using HTML, CSS, Bootstrap, and JavaScript. Recreated the dark-themed music streaming interface, including playlists, navigation, and music controls, to strengthen front-end development and UI design skills. The project is currently not fully responsive and is designed mainly for desktop viewing.

Technologies Used: HTML5, CSS3
